휴양시설 사전 경합 시뮬레이터

이번 신청서 입력 → 경합 결과 확인 → 이전 성수기 사용이력 누적 차감 계산 순서입니다. 신청 목록은 접기/펼치기로 숨길 수 있습니다.

사용 순서 : ① 이번 신청서의 이름/입사연월일/휴양지/번호를 입력합니다. ② 같은 휴양지·번호가 겹치면 경합 결과에 자동 표시됩니다. ③ 이전 휴양지 사용이력은 이름/성수기 구분/박수를 입력하면 차감점수가 자동 누적됩니다.
휴양지 번호 : 소노호텔 1~12번 / 하이원 1~13번 / 인스파이어 호텔 1~13번 / 워커힐 1~13번 / 레고랜드 1~13번
계산식 : 근속점수 = 근속개월 × 1.4점, 최대 100점 / 최종점수 = 근속점수 - 누적차감점수.
차감 기준 : 2025.12~2026.01 = 20점/박, 2025.07~08 = 15점/박, 2024.12~2025.01 = 12점/박, 2024.07~08 = 10점/박.

① 이번 신청서 입력사항

신청 건수: 0건
이번 신청 목록 펼치기 / 접기 입력 순서가 보이지 않도록 기본은 접어둘 수 있습니다.

② 경합 결과

현재 같은 휴양지/번호로 겹치는 신청이 없습니다.

③ 부서원 이전 휴양지 사용이력

이전 사용이력은 여러 건 입력 가능하며, 같은 이름의 차감점수가 자동으로 합산됩니다. 이번 신청서의 성명과 이름이 같으면 최종점수에 반영됩니다.
사용이력: 0건 / 등록 인원: 0명

사용이력 상세

사용 가능한 성수기구분: 2025 겨울, 2025 여름, 2024 겨울, 2024 여름
예: 김철수,2025 여름,2 → 15점 × 2박 = 30점 차감
